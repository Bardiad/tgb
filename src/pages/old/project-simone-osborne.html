what i did

Royal LePage, Canada's biggest real estate brokerage had asked Alaia to assist in the implementation of a new site design they had commissioned from Plastic Mobile. As lead front end designer my role involved translating Plastics PDF mockups into tangible front end code. This included creating a custom gulp build system for compiling Sass code down into CSS, as well as minifying and packing the included javascript modules, plugins and custom code
